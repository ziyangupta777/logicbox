# LogicBox

A simple Python program that provides a menu-driven interface for basic operations like generating patterns and analyzing numbers.

## Features

- **Pattern Generator**: Create asterisk patterns based on the number of rows specified.
- **Number Analyzer**: Analyze numbers in a given range, identifying even/odd numbers and calculating their sum.

## Installation

1. Ensure you have Python installed on your system (version 3.x recommended).
2. Clone or download this repository.
3. Navigate to the project directory.

## Usage

Run the script using Python:

```bash
python logicbox.py
```

Follow the on-screen menu to select an option:
1. Pattern: Enter the number of rows to generate a pattern.
2. Number Analyzer: Enter start and end values to analyze numbers in that range.
3. Exit: Quit the program.

## Example

### Pattern Generation
Enter choice: 1
Enter rows: 5

Output:
```
*
* *
* * *
* * * *
* * * * *
```

### Number Analysis
Enter choice: 2
Enter start: 1
Enter end: 5

Output:
```
1 is Odd
2 is Even
3 is Odd
4 is Even
5 is Odd
Total Sum = 15
```
